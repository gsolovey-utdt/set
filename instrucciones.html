Instrucciones de Set

Guía detallada para jugar la versión clásica.

Volver al juego

Objetivo

Encontrar grupos de 3 cartas que formen un set. Para que tres cartas sean set, en cada atributo se debe cumplir una de estas dos condiciones:

  • los 3 valores son iguales, o
  • los 3 valores son distintos.

Ejemplos visuales

Ejemplo A: set válido por cantidad

En este ejemplo, color, forma y fondo se mantienen iguales; solo cambia la cantidad (1, 2 y 3), que es todo distinto.

✅ Es set.

Ejemplo C: set válido más difícil (cambian varios atributos)

Este caso es más exigente porque en cada carta cambian varios atributos al mismo tiempo. Aun así es set, porque en cada atributo se cumple "todo igual" o "todo distinto": color (3 distintos), forma (3 distintas), fondo (3 distintos) y cantidad (1, 2, 3).

Lila, rombo, sólido, 1

Verde, óvalo, rallado, 2

Rojo, cuadrado, sin fondo, 3

✅ Es set (difícil).

Atributos de cada carta

  • Color: lila, verde, rojo.
  • Forma: óvalo, rombo, cuadrado.
  • Cantidad: 1, 2 o 3 símbolos.
  • Fondo: rallado, sólido o sin fondo.

Cómo jugar en esta app

  • Hacé clic en 3 cartas para proponer un set.
  • Si es correcto, sumás 1 punto y las nuevas cartas entran en los mismos lugares.
  • Si no es correcto, se resta 1 punto (sin bajar de 0).
  • El tablero intenta mantenerse en 12 cartas cuando hay cartas en el mazo.

Botones

  • Nuevo juego: reinicia mazo, tablero, puntaje y tiempo.
  • Pista: marca una carta que pertenece a algún set visible.
  • Agregar 3 cartas: agrega 3 cartas extra desde el mazo.

Consejos útiles

  • Elegí dos cartas y deducí la tercera necesaria por atributo.
  • Si dos cartas tienen el mismo valor en un atributo, la tercera debe repetir ese valor.
  • Si dos cartas tienen valores distintos en un atributo, la tercera debe tener el tercer valor posible.
